# Bootcamp Klabin - Excel e Power BI Dashboards 2026

Este repositório contém os projetos, desafios de código e scripts desenvolvidos durante o Bootcamp de Análise de Dados da DIO em parceria com a Klabin (Carga horária: 90h).

O foco destas entregas é a aplicação prática do ciclo de vida dos dados: desde a extração e modelagem até a criação de dashboards interativos para tomada de decisão. O objetivo é unir uma forte visão analítica (Business Intelligence) com a base técnica de Engenharia de Software.

## Tecnologias e Ferramentas

* **Bancos de Dados & Processamento:** MySQL, SQL (Queries, Joins, Agrupamentos), Modelagem Dimensional (Star Schema).
* **Análise e Visualização:** Power BI, Excel, Power Query, Fórmulas DAX.
* **Inteligência Artificial:** Microsoft Copilot aplicado à análise de dados.
* **Conceitos Aplicados:** Processos de ETL, Storytelling com dados, M.I.S. (Management Information Systems).
