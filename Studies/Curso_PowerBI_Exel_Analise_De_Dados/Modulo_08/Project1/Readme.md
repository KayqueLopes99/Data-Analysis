# Desafio de Projeto: Atualização de Relatório Financeiro com Foco em UX
Este repositório contém a evolução de um relatório financeiro no Power BI, onde o foco saiu apenas da exibição de dados para a criação de uma **experiência de usuário (UX)** fluida e profissional.

##  Objetivo do Desafio

O objetivo foi reformular um relatório criativo anterior, aplicando conceitos de design para melhorar a tomada de decisão. As modificações focaram em:

* **Posicionamento**: Organização dos visuais seguindo a hierarquia de importância.

* **Contraste**: Uso de cores para destacar Kpis e diferenciar métricas de Receita e Lucro.

* **Navegabilidade**: Implementação de botões para transição suave entre as páginas do relatório.

### 2. Profit Detail Report (Detalhamento de Lucro)

* **Distribuição Geográfica**: Uso de árvore de decomposição para analisar o lucro por país, com a Alemanha liderando ($1,11 Mi).


* **Performance por Produto**: Gráfico de radar comparando o lucro entre diferentes linhas como Velo, Amarilla e Montana.


* **Análise Temporal**: Gráfico de cascata (Waterfall) exibindo o lucro acumulado por trimestre (Qtr 1 a Qtr 4).



### 3. Detailed Sales Report (Análise de Período)
* **Evolução Temporal**: Gráfico de área comparando a "Soma de Sales" e "Soma de Profit" ao longo dos meses.

* **Tendência de Vendas**: Gráfico de colunas com linha de tendência mostrando o crescimento de vendas brutas, atingindo pico em Outubro ($22 Mi).



## 🛠️ Funcionalidades Técnicas Aplicadas
* **Menu de Navegação**: Botões interativos para alternar entre as visões de Vendas e Lucro.

* **Matriz de Descrição**: Tabelas detalhadas por trimestre e ano (2013-2014) para conferência de valores exatos.

* **Filtros de Segmentação**: Segmentadores de dados por Ano e País para tornar o relatório dinâmico.



## Como Visualizar

1. Baixe o arquivo `.pbix` presente neste repositório.
2. Abra no Power BI Desktop.
3. Navegue pelas páginas utilizando os botões do menu lateral/inferior.
