#### O que é Clusterização de dados?
- Agrupamento automático de dados em grupos (clusters) com características similares
- Identifica padrões e relações entre dados sem classificação pré-definida
- No Power BI: use o visual "Clustering" ou técnicas de IA para segmentar clientes, produtos ou comportamentos
- Aplicações: segmentação de mercado, análise de comportamento do cliente, agrupamento de produtos similares
- Exemplo: agrupar clientes por padrão de compra para estratégias de marketing direcionadas
