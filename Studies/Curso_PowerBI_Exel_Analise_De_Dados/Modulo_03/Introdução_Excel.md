## Introdução ao Excel 365
O **Microsoft Excel** é um software de planilhas eletrônicas que utiliza linhas e colunas para organizar, calcular e analisar dados, criar gráficos e automatizar tarefas.
- Organização e Coordenação. 


> Termos proficionais:
- Pasta de Trabalho (Workbook): Arquivo do excel.
- Planilha exel (Worksheet): Abas abaixo/ folha de trabalho.
- Células (Cells): Cruzamento de linha e coluna / podem ser referenciadas.
- Intervalo (Ranges): Conjunto de células selecionadas.

> Meios
- "cell1;cell2;...;cellN": Células especificas. 
- ":": De uma célula a outra (ATÉ).
- Ambos ": ;": Combinação de intervalos (E.)
- Formatação.


> Menus & Context Menus
- As principais Faixa de Opções 

- Dividida em **Guias (Abas)**, e cada aba é dividida em **Grupos** de comandos relacionados.

* **Arquivo (File / Backstage View):** É o único menu que tira você da grade da planilha. Serve para gerenciar o documento como um todo (Salvar, Abrir, Imprimir, Compartilhar) e acessar as **Opções** avançadas do Excel.

* **Página Inicial (Home):** Contém os comandos mais frequentes.
* *Foco:* Formatação de texto (fonte, cor), alinhamento, formatação de números (moeda, porcentagem) e a importantíssima **Formatação Condicional**.


* **Inserir (Insert):** Usado para adicionar elementos visuais e analíticos à planilha.
* *Foco:* **Tabelas Dinâmicas (PivotTables)**, Gráficos de todos os tipos, Ilustrações e Segmentação de Dados (Filtros visuais).


* **Layout da Página (Page Layout):** Controla a estrutura visual da "folha" para impressão.
* *Foco:* Margens, orientação (retrato/paisagem), área de impressão e temas de cores.


* **Fórmulas (Formulas):** O "cérebro" do Excel.
* *Foco:* Biblioteca de Funções (separada por categorias como Lógica, Texto, Financeira), Gerenciador de Nomes e ferramentas de **Auditoria de Fórmulas** (rastrear de onde um erro está vindo).


* **Dados (Data):** **A aba mais importante para um Analista de Dados.**
* *Foco:* Importação de dados (Power Query / Obter Dados), Classificação, Filtros avançados, Validação de Dados (criar listas suspensas) e Texto para Colunas.


* **Revisão (Review):** Ferramentas de colaboração e segurança.
* *Foco:* Verificação ortográfica, inserção de comentários e **Proteger Planilha/Pasta de Trabalho** (colocar senha nas células).


* **Exibir (View):** Altera como você enxerga a tela no seu monitor.
* *Foco:* Congelar Painéis (manter o cabeçalho fixo ao rolar a página), zoom e visualização de quebra de página.

> Overview de células
- *Formulas*: financeiras, lógicas, texto, data e hora, busca e pesquisa. metemática e etc. 
- 
```
=NOME_DA_FUNCAO(argumento1; argumento2; [argumento_opcional])
```


> Cells Freezing
- Quando você arrasta uma fórmula para baixo ou para o lado, o Excel é programado para ser "inteligente" e mover as células referenciadas na mesma direção (Referência Relativa). O "Congelamento" serve para **travar** essa movimentação.

- A tecla para isso é o **`F4`**. Ao selecionar o endereço de uma célula na barra de fórmulas e apertar `F4`, o Excel insere o símbolo de cifrão (`$`), que funciona como um "cadeado".
