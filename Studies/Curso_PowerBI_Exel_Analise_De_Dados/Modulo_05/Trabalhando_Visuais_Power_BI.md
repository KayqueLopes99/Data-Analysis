## Explorando Dados e Relatórios com Power BI
- Power Bi desktop:  Ele é usado para criar relatórios e visualizações de dados.
- Power service: Ele é usado para compartilhar e colaborar em relatórios e visualizações de dados.

> Ajustando Relatório para criar um Layout Mobile
- No Power BI Desktop, vá para a guia "Exibição" e selecione "Layout Mobile". Isso permitirá que você ajuste o layout do seu relatório para se adequar a dispositivos móveis.
- Você pode arrastar e soltar os elementos do relatório para reorganizá-los e garantir que eles sejam exibidos corretamente em telas menores.
- Certifique-se de testar o layout em diferentes dispositivos móveis para garantir que ele seja responsivo e fácil de usar.

- **Conjunto de dados**: Ele é a fonte de dados que você usa para criar suas visualizações. Pode ser uma tabela, um arquivo Excel, um banco de dados ou uma conexão com um serviço online.

- Relatórios
- Painéis

#### Graficos e Visualizações

> Grafico de barras:
-  Ele é útil para comparar diferentes categorias ou mostrar a distribuição de dados em diferentes grupos.
- Hierarquia: Ele é útil para mostrar a relação entre diferentes níveis de dados, como categorias e subcategorias.

- Na vertical: Ele é útil para mostrar a distribuição de dados em diferentes grupos ou categorias, como vendas por região ou por produto.
- Na horizontal: Ele é útil para comparar diferentes categorias ou mostrar a distribuição de dados em diferentes grupos, como vendas por região ou por produto.


> Gráfico de linhas:
- Ele é útil para mostrar tendências ao longo do tempo, como vendas mensais ou crescimento de receita.


> Gráfico de áreas:
- Ele é útil para mostrar a contribuição de diferentes categorias para um total ao longo do tempo, como vendas por região (localização) ou por produto.
- Maginitude e intensidade: Ele é útil para mostrar a magnitude e a intensidade de diferentes categorias ou grupos, como vendas por região ou por produto.


> Gráfico Treemap:
- Ele é útil para mostrar a hierarquia e a proporção de diferentes categorias ou grupos, como vendas por região ou por produto.
- Cada retângulo representa uma categoria ou grupo, e o tamanho do retângulo é proporcional à magnitude ou intensidade da categoria ou grupo. Para grandes valores. Coloração.


> Grafico agrupado / de combinação (barras e linhas):
- Ele é útil para comparar diferentes categorias ou mostrar a distribuição de dados em diferentes grupos, como vendas por região ou por produto, e também para mostrar tendências ao longo do tempo, como vendas mensais ou crescimento de receita. Proporções.


> Cartão: 
- Ele é útil para mostrar um único valor ou métrica, como vendas totais ou lucro líquido. Ele pode ser formatado para destacar o valor, como usando uma cor de fundo ou um ícone. Totalidade.



> Graficos em cascata:
- Ele é útil para mostrar a contribuição de diferentes categorias para um total, como vendas por região ou por produto, e também para mostrar a variação de um valor ao longo do tempo, como vendas mensais ou crescimento de receita. Variação. proporção. Comparação.


> Grafico de rosca:
- Ele é útil para mostrar a proporção de diferentes categorias ou grupos em relação a um total, como vendas por região ou por produto. Ele é semelhante a um gráfico de pizza, mas com um espaço no centro, o que pode facilitar a leitura e a comparação de categorias ou grupos. Proporção. Comparação.


> Gráfico de pizza:
- Ele é útil para mostrar a proporção de diferentes categorias ou grupos em relação a um total, como vendas por região ou por produto. Ele é semelhante a um gráfico de rosca, mas sem um espaço no centro, o que pode dificultar a leitura e a comparação de categorias ou grupos. Proporção. Comparação.


> Árvore hierárquica:
- Ele é útil para mostrar a hierarquia e a proporção de diferentes categorias ou grupos, como vendas por região ou por produto. Ele é semelhante a um gráfico de treemap, mas com uma estrutura de árvore, onde cada nó representa uma categoria ou grupo, e o tamanho do nó é proporcional à magnitude ou intensidade da categoria ou grupo. Hierarquia. Proporção.


> Segmentação de dados:
- Ele é útil para filtrar os dados em um relatório ou painel, permitindo que os usuários selecionem categorias ou grupos específicos para exibir nas visualizações. Ele pode ser usado para criar uma experiência interativa para os usuários, permitindo que eles explorem os dados de forma mais detalhada. Interatividade. Filtro.

> Gráfico de dispersão:
- Ele é útil para mostrar a relação entre duas variáveis, como vendas e lucro, ou para identificar padrões ou tendências em um conjunto de dados, como a relação entre idade e renda. Ele pode ser usado para identificar correlações, outliers ou clusters em um conjunto de dados. Correlação. Padrões.


> Mapa:
- Ele é útil para mostrar a distribuição geográfica de dados, como vendas por região ou por país. Ele pode ser usado para identificar padrões ou tendências em um conjunto de dados, como a relação entre localização e vendas. Geografia. Padrões.

> indicador:
- Ele é útil para mostrar um valor ou métrica em relação a um objetivo ou meta, como vendas atuais em relação a uma meta de vendas. Ele pode ser formatado para destacar o valor, como usando uma cor de fundo ou um ícone, e pode incluir um indicador visual, como uma seta ou um semáforo, para mostrar se o valor está acima, abaixo ou dentro da meta. Metas. Desempenho.

> Gráfico de funil:
- Ele é útil para mostrar a progressão de um processo ou funil, como o processo de vendas ou o processo de conversão de leads. Ele pode ser usado para identificar gargalos ou áreas de melhoria em um processo, como a taxa de conversão em cada etapa do funil. Processos. Melhoria. Maior para menor.


#### Criando um Dica de Ferramenta Paginada
- Pode-se criar uma dica de ferramenta paginada no Power BI usando a funcionalidade de "Dica de Ferramenta" (Tooltip). Para fazer isso, siga os passos abaixo:
1. No Power BI Desktop, crie um relatório ou visualização que você deseja usar como dica de ferramenta.
2. Selecione o visual ou relatório que você deseja usar como dica de ferramenta e vá para a guia "Visualizações" no painel direito.
3. Na seção "Dica de Ferramenta", ative a opção "Usar como dica de ferramenta".
4. Personalize a dica de ferramenta conforme necessário, como ajustar o layout, adicionar campos ou configurar a formatação.
5. Salve o relatório ou visualização e publique-o no Power BI Service.
6. Agora, quando os usuários passarem o mouse sobre o visual ou relatório que você configurou como dica de ferramenta, eles verão a dica de ferramenta paginada que você criou, exibindo informações adicionais ou detalhes relacionados ao visual ou relatório principal.

#### Explorando Visuais Personalizados com Power BI
- Image grid: Ele é útil para mostrar uma coleção de imagens ou fotos, como um portfólio de produtos ou um álbum de fotos. Ele pode ser personalizado para exibir as imagens em diferentes layouts, como uma grade ou uma lista, e pode incluir recursos interativos, como a capacidade de clicar em uma imagem para obter mais informações ou para navegar para outra página. Imagens. Personalização.

- World clound: Ele é útil para mostrar a frequência ou a importância de diferentes palavras ou termos em um conjunto de dados, como uma análise de sentimentos ou uma análise de texto. Ele pode ser personalizado para exibir as palavras em diferentes tamanhos ou cores, com base na frequência ou na importância das palavras. Análise de texto. Personalização.
