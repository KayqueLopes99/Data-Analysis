## Gateway

Conectividade e atualização

### Tipos

- **Modo de organização**: vários usuários e fontes
- **Modo pessoal**: usuário único

O Gateway de Dados do Power BI é um software que atua como uma ponte segura entre os dados locais (on-premises), como arquivos Excel no seu computador ou bancos de dados internos (SQL Server), e o Power BI Service na nuvem. Ele é fundamental para atualizar relatórios automaticamente e manter os dados sincronizados.

## Clusters no Power BI

Clusters no Power BI são agrupamentos automáticos de dados semelhantes, utilizando algoritmos de machine learning (geralmente k-means) para segmentar informações em gráficos de dispersão ou mapas. Eles facilitam a identificação de padrões, comportamentos ou semelhanças ocultas (ex: clientes por padrão de compra) sem necessidade de fórmulas complexas.
